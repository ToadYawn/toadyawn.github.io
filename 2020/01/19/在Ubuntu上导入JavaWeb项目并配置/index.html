在Ubuntu上导入JavaWeb项目并配置
发布于：2020-01-19 04:02:35 访问：

准备ubuntu基本环境

  • 安装 JDK 任意版本
    一搬安装在 /opt/java/ 解压

    配置环境变量 /etc/profile

    使之生效

    检查之

  • 安装 eclipse 任意版本
    一般安装在 /opt/eclipse/ 解压

    建立软连接，使 jre 成为其子目录

添加 eclipse 桌面图标

准备项目包

配置eclipse环境

  • 设置好工作空间
    一般也在 /opt/eclipse_workspace
  • 指定项目内的 jre 为新项目的 build path
    在 preference>Java>Installed JREs 中 add 路径为 /opt/kingkong_web/local/jdk1.7.xxxx
  • 导入项目包
    import existing project into workspace
    选择目录为 /opt/kingkong_web/ 后 finish 导入
  • 设置项目内 tomcat 作为应用服务器
    打开 server view(windows>show views>server)
    新建 server, 选 tomcat 7.0 以及 /opt/kingkong_web/apache-tomcar-7.0.37, 将 KKWeb add 配置到上面 finish

配置环境变量以及启动 mysql/redis

  • 获取当前 user

  • 获取 shell 文件所在 bin 父文件夹的绝对路径（即项目文件夹）

  • 修改 PATH 可执行程序的查找路径

  • 修改 LD_LIBRARY_PATH 动态库的查找路径

  • 根据start/stop/status运行响应配置程序

在 eclipse 中启动 tomcat 即可/或者直接 start 中就可以
